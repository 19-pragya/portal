Welcome to Systers Portal Project

Systers strive to help women improve or develop their technical skills so they are contributors in open source and other technical communities. We have several projects we work on as a community as well as part of Google Summer of Code and Google Code In (melange).

Check out the new branch:

$ cd your_repo_root/repo_name
$ git fetch origin
$ git checkout gh-pages

If you're using the GitHub for Mac, simply sync your repository and you'll see the new branch.

The Problem

Before rebranding and new website, ABI (Anita Borg Institute) had a section/pages for Systers and each subgroup (Latinas, LGBT, Black Women in Computing, TurkishWiC and ArabWiC). This allowed for subgroup members to find information particular to their community or main Systers community.

Approach

The portal project is to have admins from Systers and subgroups to provide the same information above in one portal area. A portal for Systers and sub-groups to share information to perspective communities, lists events and by geography.
